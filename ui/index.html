How It Works — 5-Agent Pipeline
01
Intake
Polls Microsoft Sentinel for new incidents via REST API, parses entities (IPs, accounts, hosts), and fetches full incident details including correlated alerts.
→
02
Enrichment
Enriches every entity: VirusTotal + AbuseIPDB + GreyNoise for IPs (scanner/malicious/legit classification), UEBA behavioral priority (0–10) for users, Foundry IQ IOC reputation for hashes.
→
03
Investigation
Classifies the incident (TruePositive / FalsePositive / Benign / Escalate) and maps to MITRE ATT&CK tactics via Foundry IQ — Azure AI Search index of 47 techniques. No hallucination: every technique cite is grounded in a real index entry.
→
04
Response
Drafts containment actions (IP block, device isolation, account disable) and produces a full incident report with timeline, IOC list, and confidence score — ready for human review.
→
05
ECHO
Synthesizes a new KQL detection rule from the confirmed attack pattern and pushes it to Microsoft Sentinel. The next identical attack is caught automatically — no analyst needed.
